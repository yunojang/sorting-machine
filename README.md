# preonboarding Assignment 6
솔라커넥트 과제 Sorting Machine

### 소개
* **팀원** : 강준희, 장윤호
* **배포 주소** : https://dreamy-engelbart-37b1eb.netlify.app/

### 기능 구현
* **타이머** : 날짜, 시간을 확인 합니다.
* **정렬** : 숫자를 입력 받고, 정렬하여 나타냅니다.

### 팀원 별 역할
* **강준희** : 타이머, 프로젝트 레이아웃
* **장윤호** : 정렬, 문서 및 배포

### 프로젝트 설치
```
npm install
npm start
```
